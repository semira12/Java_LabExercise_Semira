
# MyNotePad

A simple JavaFX notepad application built with Java and Maven.

## Features
- Create and edit text
- Simple JavaFX interface
- Maven project structure

## Technologies
- Java
- JavaFX
- Maven

## Project Structure

```text
src/main/java/com/mynotepad/mynotepad/
 ├── Launcher.java
 └── TheNotepad.java
```

# How to Run

 1. Open the project in IntelliJ IDEA
 3. Load Maven dependencies
 4. Run Launcher.java
